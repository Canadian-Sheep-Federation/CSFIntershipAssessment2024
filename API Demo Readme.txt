API DEMO Readme:

Uses the public Useless Facts API to fetch a random useless fact.
Uses a custom API for submitting a survey form and saving the responses to a SQLite database, and retrieve responses.
Webapp as a jsp file.

What's Missing:
- A way to display the survey responses graphically
- Setting the ports for the webapp and web service so they can be used together

What could be added:
- a way to edit responses
- authentication, so users can only alter their own responses
- a filter for responses

Deployment:
- Tomcat
