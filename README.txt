A simple RestFul API that utilizes the Itunes Song API allowing clients to
- search for songs
- add and remove songs from a playlist
- view and create reviews for a song. 

INSTALL INSTUCTIONS:
>npm install

This will install the modules listed as dependencies in the package.json file.

LAUNCH INSTRUCTIONS:
>npm start
or 
>node server.js

TESTING INSTRUCTIONS URL:
    http://localhost:3000
